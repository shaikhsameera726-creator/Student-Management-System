# 🎓 Student Management System

A web-based Student Management System developed using **Java, JSP, Servlets, JDBC and PostgreSQL**.

## Features

* ➕ Add Student
* 📋 View Students
* ✏️ Update Student
* ❌ Delete Student

## Technologies Used

* Java
* JSP
* Servlets
* JDBC
* PostgreSQL
* HTML
* CSS
* Apache Tomcat

## Database

Database: PostgreSQL

Table:

```
students(
id,
name,
email,
course
)
```
